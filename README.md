# VANGUARD Gaming Store Landing Page

A Vite + React + Tailwind landing page for a futuristic gaming-store experience.

## Run locally

```bash
npm install
npm run dev
```

## Deploy for free

1. Create a GitHub repository.
2. Push the project:

```bash
git remote add origin https://github.com/<your-username>/<your-repo>.git
git branch -M main
git push -u origin main
```

3. Import the repository into Vercel or Netlify for free hosting.
